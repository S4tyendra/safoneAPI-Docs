# SafoneAPI

Asynchronous Python Wrapper For SafoneAPI

## Installation

```sh
pip install -U safoneapi
```

## Usage

For example, to search on GitHub:

```py
import asyncio
from SafoneAPI import SafoneAPI

async def main():
    api = SafoneAPI()
    resp = await api.github("AsmSafone")
    print(resp.results)

asyncio.run(main())
```

## Documentation

Take help from the docstrings this way:

```py
from SafoneAPI import SafoneAPI

print(help(SafoneAPI.logo))
```

For more, check out the [API Playground](https://api.safone.dev/docs) or [Read Docs](https://api.safone.dev/redoc).
