## Endpoint: `/apps`

### Method: `GET`

#### Parameters

- **query** (required): The query string for your search.
- **limit** (optional): Limit the number of results (default is unspecified).

### Responses

- **200 - Success**
- Returns the requested information.

- **422 - Validation Error**
- Indicates a validation error in the request.

## Example Usage

To search for apps using the Apps API, you can use the following Python code:

```python
import asyncio
from SafoneAPI import SafoneAPI

async def main():
    api = SafoneAPI()
    query = "your_search_query_here"
    limit = 10  # Optional: specify the limit

    try:
        # Make the Apps API request
        response = await api.apps(query, limit=limit)

        # Access the results
        results = response.results
        print(results)

    except Exception as e:
        # Handle errors
        print(f"Error: {e}")

# Run the example
asyncio.run(main())
```
